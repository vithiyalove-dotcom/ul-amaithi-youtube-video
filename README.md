# Ul Amaithi – YouTube Video Automation

This repository contains FFmpeg automation scripts used to create
cinematic YouTube videos for the Tamil channel "உள் அமைதி".

## Files
- clips.txt : List of video clips in order
- run.bat   : FFmpeg script to merge clips with audio

## Usage
1. Keep video clips inside a local `clips/` folder
2. Keep audio file as `audio.m4a`
3. Run `run.bat` on your PC
4. Output video will be generated as `output.mp4`

Note: Video and audio files are not uploaded to GitHub due to size limits.
